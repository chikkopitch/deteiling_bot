# Руководство администратора

Telegram ID администратора должен быть в `ADMIN_IDS`. Команда `/admin` открывает заявки. Карточка позволяет подтвердить или отклонить запись; каждое решение проходит серверную проверку роли и попадает в `audit_logs`. Подтверждение создаёт напоминания, отклонение освобождает слот. Контент хранится в таблицах услуг, FAQ, правил цены и расписания; массовые изменения безопаснее выполнять через отдельную проверенную административную миграцию. Не удаляйте услуги физически, если они участвуют в записи — используйте `is_active`/`deleted_at`.
