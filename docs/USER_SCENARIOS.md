# Сценарии клиента

Нажмите «Записаться», выберите марку/модель, год, класс и услуги. Фото можно пропустить. Выберите свободное время, укажите имя и телефон, проверьте данные и отправьте заявку. Слот удерживается ограниченное время. После решения администратора придёт сообщение; сведения доступны в «Моя запись». Каталог показывает цены «от», калькулятор — ориентировочный диапазон. FAQ отвечает на частые вопросы, а «Связаться с менеджером» сохраняет обращение и контакт.
