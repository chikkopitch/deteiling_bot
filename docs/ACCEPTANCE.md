# Приёмка

Автоматически проверяются конфигурация, телефоны, меню, upsert пользователя, активный каталог/FAQ, state machine, удержание слота и идемпотентная отправка. Ручная приёмка требует реального Telegram sandbox: `/start`, полный FSM с фото, получение карточки администратором, подтверждение/отклонение, доставка напоминаний, калькулятор и менеджер. Инфраструктурная приёмка: healthchecks зелёные, миграция upgrade/downgrade на тестовой БД, сохранение volumes после restart и восстановление backup.
