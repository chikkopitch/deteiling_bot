# Развёртывание

Используйте отдельный Linux-хост, TLS-прокси не требуется для polling. Храните `.env` вне репозитория, ограничьте права файла, задайте уникальный пароль PostgreSQL и закрепите версии образов после проверки. Запуск: `docker compose up -d --build`; статус: `docker compose ps`; логи: `docker compose logs --tail=200 bot`. Перед обновлением сделайте backup, соберите образ, примените миграцию и проверьте `/start`. Для высокой доступности запускайте один polling-инстанс или переходите на webhook с координацией.
